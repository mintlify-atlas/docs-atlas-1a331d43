## System requirements

| Requirement | Value |
|---|---|
| macOS | 10.14 (Mojave) or later |
| RAM | 4 GB minimum, 8 GB recommended |
| Storage | 100 MB |
| Microphone | Required |
| Internet | Required for AI transcription |

## Option 1: Download DMG (recommended)

The DMG release is the simplest way to install AutoQuill. It is signed and notarized for macOS Gatekeeper compatibility.

<Steps>
  <Step title="Download the latest release">
    Go to the [AutoQuill AI releases page](https://github.com/DevelopedByDev/autoquill_ai/releases) and download the latest `AutoQuill-v{version}.dmg` file.
  </Step>
  <Step title="Open the DMG">
    Double-click the downloaded `.dmg` file to mount it. A Finder window will appear showing the AutoQuill app icon and an **Applications** folder shortcut.
  </Step>
  <Step title="Drag to Applications">
    Drag the **AutoQuill** icon onto the **Applications** folder shortcut inside the DMG window.
  </Step>
  <Step title="Launch AutoQuill">
    Open **Finder → Applications** and double-click **AutoQuill** to launch it.

    <Note>
      On first launch, macOS may show a security prompt. Click **Open** to proceed. If the app is blocked, go to **System Preferences → Privacy & Security** and click **Open Anyway**.
    </Note>
  </Step>
  <Step title="Complete onboarding">
    AutoQuill will guide you through granting permissions, adding your Groq API key, and configuring hotkeys. See [Quickstart](/quickstart) for a walkthrough.
  </Step>
</Steps>

## Option 2: Build from source

Building from source requires Flutter and Xcode installed on your Mac. Use this method if you want to run a development build or contribute to the project.

### Prerequisites

<Steps>
  <Step title="Install Flutter">
    Follow the [official Flutter installation guide](https://docs.flutter.dev/get-started/install) for macOS. AutoQuill requires Flutter 3.1.3 or later.

    After installation, verify your setup:

    ```bash
    flutter doctor
    ```

    Resolve any issues Flutter doctor reports before continuing.
  </Step>
  <Step title="Install Xcode">
    Install Xcode from the Mac App Store, then install the command line tools:

    ```bash
    xcode-select --install
    ```
  </Step>
  <Step title="Install CocoaPods">
    CocoaPods is required for macOS-specific dependencies:

    ```bash
    sudo gem install cocoapods
    ```
  </Step>
</Steps>

### Build steps

<Steps>
  <Step title="Clone the repository">
    ```bash
    git clone https://github.com/DevelopedByDev/autoquill_ai.git
    cd autoquill_ai
    ```
  </Step>
  <Step title="Install Flutter dependencies">
    ```bash
    flutter pub get
    ```
  </Step>
  <Step title="Install CocoaPods dependencies">
    ```bash
    cd macos && pod install && cd ..
    ```
  </Step>
  <Step title="Generate app icons">
    ```bash
    flutter pub run flutter_launcher_icons
    ```
  </Step>
  <Step title="Run the app">
    ```bash
    flutter run -d macos
    ```

    For a release build:

    ```bash
    flutter build macos --release
    ```
  </Step>
</Steps>

<Tip>
  If you encounter build errors, clean the build environment and reinstall dependencies:

  ```bash
  flutter clean
  rm -rf macos/Pods
  cd macos && pod install && cd ..
  flutter pub get
  flutter build macos
  ```
</Tip>

## Auto-updates

The DMG release includes the Sparkle auto-update framework. AutoQuill checks for updates on startup and applies them in the background. You do not need to manually reinstall to stay up to date.

<Note>
  Auto-updates are only available in the signed DMG release. Development builds compiled from source do not receive automatic updates.
</Note>

## Next steps

Once AutoQuill is installed, follow the [Quickstart](/quickstart) guide to grant permissions, add your API key, and make your first transcription.
