title: "Quickstart"
description: "Go from a fresh AutoQuill installation to your first transcription in five steps."
---

This guide assumes you have already installed AutoQuill. If not, see [Installation](/installation) first.

<Steps>
  <Step title="Launch AutoQuill and start onboarding">
    Open **AutoQuill** from your Applications folder. On first launch, AutoQuill displays the onboarding flow automatically.

    The welcome screen introduces the three core features:

    - **Transcribe Audio** — convert speech to clean text with a keyboard shortcut
    - **AI Assistant** — edit and generate text with voice instructions
    - **No login required** — you supply your own API key; no account needed

    Click **Continue** to proceed.
  </Step>
  <Step title="Grant required permissions">
    AutoQuill needs three macOS permissions to function. The onboarding **Grant Permissions** screen walks you through each one.

    | Permission | Why it is needed | How to grant |
    |---|---|---|
    | Microphone | Captures audio for transcription | Click **Grant Permission** when prompted |
    | Accessibility | Enables global hotkeys and automatic text insertion | Click **Open Settings**, then add AutoQuill under **Privacy & Security → Accessibility** |
    | Screen recording | Provides visual context for AI Assistant mode | Click **Grant Permission**; the app may need to restart |

    <Warning>
      Accessibility permission is mandatory for hotkeys to work in other applications. If you skip it, AutoQuill can only transcribe while its own window is focused.
    </Warning>

    After granting each permission, click **Check Permissions Again** to refresh the status. Once all three show **Granted**, click **Continue**.

    <Note>
      If microphone permission is denied, go to **System Preferences → Privacy & Security → Microphone**, enable AutoQuill, then restart the app.
    </Note>
  </Step>
  <Step title="Add your Groq API key">
    AutoQuill uses [Groq](https://console.groq.com) for AI-powered transcription. You need a free Groq API key to continue.

    **Get your API key:**

    1. Create a free account at [groq.com](https://console.groq.com/keys)
    2. Go to **API Keys** in your dashboard
    3. Click **Create API key**, name it `AutoQuill`, and click **Submit**
    4. Copy the key and click **Done**

    **Enter your key in AutoQuill:**

    Paste the key into the **Groq API Key** field on the **Connect to Groq** screen, then click **Validate Key**. AutoQuill confirms the key is valid before proceeding.

    <Note>
      Your API key is stored locally in the Hive database at `autoquill_persistent_data`. It is never transmitted to any server other than Groq's API endpoint during transcription.
    </Note>
  </Step>
  <Step title="Set your hotkeys">
    The **Set Your Shortcuts** screen shows the default hotkeys for each mode. You can keep the defaults or record new combinations now.

    | Mode | Default hotkey |
    |---|---|
    | Standard transcription | `⌥ Shift Z` (Alt/Option + Shift + Z) |
    | Push-to-Talk | `⌥ Space` (Alt/Option + Space) |
    | AI Assistant | `⌥ Shift S` (Alt/Option + Shift + S) |

    To change a hotkey, click the **edit** icon next to any shortcut and press your preferred key combination. AutoQuill detects conflicts between modes automatically.

    You can also enable **Smart Transcription** here — it adds intelligent formatting, emoji conversion, and proper casing to your transcription output. It is slower than standard transcription but produces more polished results.

    <Tip>
      Use modifier keys (`⌥`, `⌃`, `⇧`, `⌘`) combined with a letter key for system-wide compatibility. Avoid shortcuts already used by macOS (e.g., `⌘ Space` for Spotlight).
    </Tip>

    Click **Continue** when done. AutoQuill saves your hotkeys and registers them system-wide.
  </Step>
  <Step title="Make your first transcription">
    AutoQuill is now set up. Open any application with a text field — a document, email, chat window, or code editor — and try each mode:

    **Standard transcription:**
    1. Press `⌥ Shift Z` (or your custom hotkey)
    2. Speak your message
    3. Press `⌥ Shift Z` again to stop
    4. The transcribed text is copied to your clipboard and pasted automatically

    **Push-to-Talk:**
    1. Hold `⌥ Space` while speaking
    2. Release the key to transcribe immediately
    3. Text is copied and ready to paste with `⌘V`

    **AI Assistant mode:**
    1. Select text in any application
    2. Press `⌥ Shift S`
    3. Speak your instruction (for example: "make this more formal" or "fix the grammar")
    4. The AI processes your instruction and replaces the selected text

    Press `Esc` at any point to cancel an active recording.

    <Note>
      The recording overlay appears in the corner of your screen while AutoQuill is listening. It shows the current mode, status, and a hotkey reminder. It disappears automatically after transcription completes.
    </Note>
  </Step>
</Steps>
